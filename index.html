Математичне моделювання прикладної задачі

🚀 Що таке математична модель?

Математична модель — це опис реальної ситуації мовою математичних символів (рівнянь, нерівностей, функцій). Коли ми розв'язуємо прикладну задачу, ми "перекладаємо" її з людської мови на мову рівнянь.

👣 5 кроків до розв'язку:

  1. Вибір невідомого: Позначаємо головну шукану величину через \(x\).
  2. Зв'язки: Виражаємо інші величини через \(x\).
  3. Побудова рівняння: Складаємо математичну модель ситуації.
  4. Обчислення: Знаходимо корені отриманого рівняння.
  5. Debug (Аналіз): Перевіряємо, чи підходить результат реальності (напр. швидкість не може бути від'ємною).

📐 База: Квадратні рівняння

Загальний вигляд:

\[ax^2 + bx + c = 0, \quad a \neq 0\]

1. Дискримінант (\(D\))

Від нього залежить кількість розв'язків:

\[D = b^2 - 4ac\]
  • \(D > 0\): два різні корені.
  • \(D = 0\): один корінь (або два однакові).
  • \(D < 0\): коренів немає.

2. Знаходження коренів

\[x_{1,2} = \frac{-b \pm \sqrt{D}}{2a}\]

3. Теорема Вієта

Для зведеного рівняння (\(x^2 + px + q = 0\)):

\[\begin{cases} x_1 + x_2 = -p \\ x_1 \cdot x_2 = q \end{cases}\]

💡 Типи задач

🔹 Геометрія: Використовуємо формули площі (\(S = a \cdot b\)) або теорему Піфагора.

🔹 Рух за течією: Швидкість за течією = \(V_{власна} + V_{течії}\). Проти течії = \(V_{власна} - V_{течії}\).

🔹 Робота: Формула \(A = P \cdot t\), де \(P\) — продуктивність, \(t\) — час.
